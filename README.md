# Etch a sketch!


Assignment was to create a **Etch a Sketch** app with HTML, CSS and JS.

The bonus assignment was to make it possible to draw with a random RGB value for every new square (**Rainbow mode**).

**Check it out** [here](https://michael-johansson.github.io/etch-a-sketch/)

## What I learned

* CSS Grid was definitely the move for this.
* Using JavaScript to change CSS.
* The forEach loop.
* A lot of minor things!
